Ao Bot is a simple Discord bot hosted on Replit and it's server is on UpTimeBot.
Here's the link to it in Replit https://replit.com/@NadaA-hash/AoBot-JS?v=1

Current Features:
- When discord members when they say "sad words" (ex: im sad) and replies with encouraging messages. 
- "ao hello" will prompt Ao to reply hello as well.
- "ao band4band [insert name here]" will prompt Ao to compare you and the name on who has the most bands.

More to be added soon. As well bug fixes.
